### Local setup

Dependencies:
* [Elm](https://guide.elm-lang.org/install.html)
* [Node](https://nodejs.org/en/download/)

To run locally:
```
git clone URL_HERE!!!!
cd aspect-ratio-calculator/
elm-package install
npm install -g create-elm-app
elm-app start
```

To run tests:
```
elm-app test
```
